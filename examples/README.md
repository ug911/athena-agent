# Vetted SQL examples

Each file is one query. First line is a `-- intent: ...` comment describing what the query answers. Add to this directory whenever Claude (or you) writes a non-trivial query that worked — these are the highest-signal context Claude has when authoring new SQL.

Filename convention: `<entity>_<metric>.sql`, e.g. `users_dau.sql`, `orders_revenue_by_country.sql`.
